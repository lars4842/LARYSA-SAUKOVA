       объяснение дипломного проекта

1. Комментируем основные действия проекта. Символ # решётки является признаком комментария
2. С помощью встроенной функции print() выводим текст приветствия. 
3. Создаём переменную s1. С помощью оператора присваивания (знака =) присваиваем значение переменной-
встроенную функцию input применяем для считывания данных из консоли.
Данная функция считает и запишет в переменную то, что будет напечатано в консоль. 
Здесь функция input считывает переменную типа string, то есть считывает текст.
Задаём вопрос об имени человека и получаем ответ.
4. Создаём переменную s2. С помощью этой переменной мы выводим текст-приветствие. Плюс к тексту добавляем
переменную s1, которая здесь добавит имя, напечатанное в консоли заказчиком.
 Происходит приветствие (знакомство) с человеком.
5. С помощью функции print(s2) мы выведем приветствие.В круглые скобки здесь мы подадим переменную s2.
6. Закомментировала строку решётками, так было удобнее отделить одно действие в чат-боте от другого.
7. Комментирую. Дальше будет идти предложение к заказчику по распечатке фотографий
8. print() выводим текст-обращение с предложением 1 к заказчику
9. print() выводим текст-предложение услуги 1
10.Создаём переменную s1. С помощью оператора присваивания (знака =) присваиваем значение переменной-
встроенную функцию input применяем для считывания данных из консоли. В результате диалога совершаем призыв к действию.
Человек соглашается воспользоваться услугой и ставит + (знак плюс). Функция input считает переменную. Здесь
мы должны получить согласие на предоставление услуги.
11.Комментируем символом # основные действия проекта. Далее применим калькулятор для расчёта стоимости услуги.
12. print() выводим текст-номер калькулятора. Так как услуг будет много и будет удобно чтобы на разных этапах расчёта
были разные калькуляторы. Затем они будут выводить общие суммы заказов. 
13. print() выводим текст-цена на услугу 1-по печати фото 10х15
